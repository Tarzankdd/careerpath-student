# CareerPath Student

A high-fidelity React + Vite + Tailwind CSS prototype for a student career planning platform.

## Features

- Dashboard with career goal, recommendations, progress, quick actions, and AI suggestion
- Internship tab with mock listings, filters, detail card, apply, and save
- Job tab with job listings, filters, salary range, detail card, apply, and save
- Career plan tab with goal selection, skills, courses, timeline, roadmap, and clickable checklist
- Profile / Resume tab with student profile, education, skills, experience, resume tips, and portfolio link
- Saved opportunities tab with internships, jobs, and application status tracking

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL printed by Vite, usually `http://localhost:5173/`.

## Build

```bash
npm run build
```
